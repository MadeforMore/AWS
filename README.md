# 📖 Overview

AWS Builder Center now provides eligible users with a **Free Sandbox
Environment** for selected workshops.

## Benefits

-   💳 No Credit Card Required
-   💰 Completely Free
-   ☁️ Real AWS Console
-   ⏰ 8 Hours of Access
-   🧹 Automatic Cleanup

------------------------------------------------------------------------

# 🛠️ Step-by-Step Guide

## Step 1 -- Open AWS Builder Center

1.  Visit **https://builder.aws.com**
2.  Sign in with your AWS account.
3.  From the left menu click **Workshops**.

------------------------------------------------------------------------

## Step 2 -- Request a Free Sandbox

1.  Browse the available workshops.
2.  Select a workshop marked with **Free AWS Environment Available**.
3.  Click **Request Free Sandbox Environment**.
4.  Wait a few minutes while AWS prepares your environment.

------------------------------------------------------------------------

## Step 3 -- Open Your Workshop

1.  Go to **Your Workshops**.
2.  Locate the requested workshop.
3.  Click **Go to Free Sandbox Environment**.

------------------------------------------------------------------------

## Step 4 -- Launch AWS Console

Inside the workshop:

-   Explore the lab instructions.
-   Click **Open AWS Console**.
-   A fully configured AWS account opens automatically.

------------------------------------------------------------------------

# 🎉 You're Ready!

Your sandbox already contains the required permissions and workshop
resources.

You can now practice AWS services without using your personal AWS
account.
