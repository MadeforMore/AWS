 ## 𝐇𝐚𝐧𝐝𝐬-𝐨𝐧 𝐰𝐢𝐭𝐡 𝐀𝐖𝐒 𝐋𝐚𝐦𝐛𝐝𝐚: 𝐀𝐮𝐭𝐨𝐦𝐚𝐭𝐞𝐝 𝐈𝐦𝐚𝐠𝐞 𝐏𝐫𝐨𝐜𝐞𝐬𝐬𝐢𝐧𝐠 𝐔𝐬𝐢𝐧𝐠 𝐒3 & 𝐒𝐍𝐒

<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/b0b096ce-d629-495f-8126-6ba54a79e782" />

The goal was simple: whenever an image is uploaded to an 𝐒3 𝐛𝐮𝐜𝐤𝐞𝐭, 𝐀𝐖𝐒 𝐚𝐮𝐭𝐨𝐦𝐚𝐭𝐢𝐜𝐚𝐥𝐥𝐲 𝐩𝐫𝐨𝐜𝐞𝐬𝐬𝐞𝐬 𝐢𝐭 𝐰𝐢𝐭𝐡𝐨𝐮𝐭 𝐚𝐧𝐲 𝐦𝐚𝐧𝐮𝐚𝐥 𝐰𝐨𝐫𝐤.

𝐖𝐡𝐚𝐭 𝐈 𝐃𝐢𝐝:

Created two Amazon S3 buckets

𝐒𝐨𝐮𝐫𝐜𝐞 𝐁𝐮𝐜𝐤𝐞𝐭 – for uploading images
𝐃𝐞𝐬𝐭𝐢𝐧𝐚𝐭𝐢𝐨𝐧 𝐁𝐮𝐜𝐤𝐞𝐭 – for storing processed images

Created an AWS Lambda function

Configured an S3 Event Trigger to invoke the Lambda function whenever a new image is uploaded

𝐓𝐡𝐞 𝐋𝐚𝐦𝐛𝐝𝐚 𝐟𝐮𝐧𝐜𝐭𝐢𝐨𝐧 𝐚𝐮𝐭𝐨𝐦𝐚𝐭𝐢𝐜𝐚𝐥𝐥𝐲:

* Reads the uploaded image
* Reduces (resizes/compresses) the image size
* Saves the processed image to the destination S3 bucket

𝐂𝐨𝐧𝐟𝐢𝐠𝐮𝐫𝐞𝐝 𝐀𝐦𝐚𝐳𝐨𝐧 𝐒𝐍𝐒

Received a confirmation email after the image was successfully processed and stored in the destination bucket
